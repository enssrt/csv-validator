# Проект: Валидатор структуры CSV-файлов

Учебный модульный проект для автоматической проверки заголовков и структуры CSV-файлов.

## Структура проекта
* `source/` — исходный код программы.
* `data/` — директория для входящих CSV-файлов.
* `data_reports/` — директория для сгенерированных отчетов.

## Требования
* Python 3.x
* Сторонние библиотеки не требуются.

## Запуск проекта
Для запуска процесса валидации выполните команду из корневой директории проекта:
```bash
python source/main.py
